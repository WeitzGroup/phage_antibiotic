README

Generate all the figures (and supplementary figures) of the manuscript: "Quantitative Models of Phage-Antibiotics Combination Therapy", Rodriguez-Gonzalez et al. (2019) by running the following script: 

run([pwd '/Gen_all_figs.m'])

The code_comb_therapy directory contains all the necessary MATLAB scripts to generate the figures.

Generated figures will be saved inside the 'figures' directory.
